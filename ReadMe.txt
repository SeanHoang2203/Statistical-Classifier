To run script (do one of the following):

1) Open main_script.py in IDLE and run module in python shell.

2) Open terminal, navigate to the folder containing main_script.py and run command
"python main_script.py" or
"py main_script.py" or
"python3 main_script.py" or
"py3 main_script.py"
